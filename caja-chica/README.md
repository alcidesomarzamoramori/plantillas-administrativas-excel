# Control de Caja Chica

Plantilla diseñada para llevar un control ordenado y estructurado de los movimientos de caja chica en pequeños y medianos negocios.

---
## Funcionalidades-básicas-principales
- Registro cronológico de ingresos y egresos
- Clasificación por categoría de gasto
- Cálculo automático de saldo disponible
- Resumen mensual
- Estructura lista para impresión

---
## Contenido-de-las-plantillas
- Registro de movimientos
- Resúmenes automático
- Reportes mensuales
