# plantillas-administrativas-excel
Excel templates for administrative control, financial management and business decision-making.
---
## Objetivo-de-carpeta

Facilitar herramientas reales para:
- Control de caja chica  
- Gestión de flujo de caja  
- Presupuestos empresariales  
- Control de cuentas por cobrar y pagar  
- Análisis financiero con dashboards  
---
## Estructura-del-repositorio

Cada carpeta contiene:
- Archivo Excel (.xlsx)  
- Captura de pantalla  
- Descripción funcional  
- Versión y fecha de actualización
